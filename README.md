# SinglePageHomework

A basic PLP view created with Jetpack Compose and Kotlin.

## Introduction

This project was developed as the 3rd assignment for the FMSS Android Bootcamp. 
It showcases a product page designed with Jetpack Compose and Kotlin, featuring functionalities such as displaying product images, names, prices, and ratings. 
Additionally, it includes options for adding products to the cart, providing a seamless user experience.

## Installation

1. **Clone the repository:**
   ```
   git clone https://github.com/BartugKaan/SinglePageHomework.git
   ```
2. **Open the project in Android Studio.**
3. **Build the project:**
   - Ensure you have the latest version of Android Studio.
   - Open the project and let it sync.
